# React Client

This is a React front-end for the Flask authentication tutorial. It uses [Vite](https://vitejs.dev/) for bundling.

## Setup

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Start the development server**
   ```bash
   npm start
   ```
   This runs Vite's dev server at [http://localhost:5173](http://localhost:5173).

3. **Build for production**
   ```bash
   npm run build
   ```
   The bundled files will be output to the `dist` directory.
